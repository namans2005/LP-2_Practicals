1. What is Google Cloud Platform?
👉 It is a cloud service where we can run applications, store data, and use computing resources online instead of local system.


2. What is a project in Google Cloud?
👉 A project is a workspace/container where all resources (code, VM, storage) are managed.


3. Why did you create ccAcademia project?
👉 To organize and run my practical inside a dedicated cloud environment.


4. What is Cloud Shell?
👉 Cloud Shell is a browser-based terminal provided by Google Cloud to run commands.


5. Why use Cloud Shell instead of local terminal?
👉 Because:
    no setup needed
    runs on cloud
    ready with tools like git and python


6. What is GitHub?
👉 A platform to store, manage, and share code repositories.


7. What does git clone do?
👉 It copies a repository from GitHub to your Cloud Shell environment.
